== README

CRUD API Requests:

* POST: ```curl -v -H "Accept: application/json" -H "Content-type: application/json" -X POST -d '{"article":{"title":"hello world","content":"this is content","user_id":"1"}}' http://localhost:3000/api/v1/articles```

* GET: ```curl -i -H "Accept: application/json" -H 'Accept: application/json' http://localhost:3000/api/v1/articles```

* PUT: ```curl -H 'Content-Type: application/json' -H 'Accept: application/json' -X PUT -d '{"article":{"title":"hello world","content":"this is content","user_id":"1"}}' http://localhost:3000/api/v1/articles/4```

* DELETE: ```curl -i -H "Accept: application/json" -X DELETE http://localhost:3000/api/v1/articles/1```
